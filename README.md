# E-Commerce Sales Analytics

## Project Overview
This project is an **E-Commerce Sales Analytics System** built using **Python and Jupyter Notebook**.  
It analyzes e-commerce sales data stored in CSV files and converts raw data into meaningful insights using **SQL queries and data visualization**.

The entire project is executed **only inside Jupyter Notebook**, without using any external tools.

--- 

## Project Goals
- Analyze total sales and order trends
- Understand customer purchasing behavior
- Identify popular payment modes
- Perform SQL-based analytics inside Jupyter Notebook
- Present data using clear graphs

---

## Tools & Technologies Used
- **Python**
- **Jupyter Notebook**
- **Pandas** – data handling
- **SQLite** – database
- **SQL** – data analysis
- **Matplotlib** – data visualization

---

## Dataset Description
The project uses the following CSV files:

- `customers.csv` – customer details  
- `products.csv` – product information  
- `orders.csv` – order records  
- `payments.csv` – payment details  

These files are loaded into a SQLite database for analysis.

---

## Project Workflow
1. Load CSV files using Pandas  
2. Store data into SQLite database  
3. Perform SQL queries for analysis  
4. Analyze total sales and order trends  
5. Visualize data using charts  

---

## Key Analysis Performed

### Total Sales Analysis
- Calculates total revenue generated from all orders
### Monthly Revenue (Bar Chart)
- Shows total revenue for each month.
- Helps identify which month has highest sales.

### Top Products for Revenue (Bar Chart)
- Shows top selling products based on revenue.
- Helps identify which products generate the most profit.

### Payment Mode Distribution (Pie Chart)
- Shows the percentage of customers using each payment mode.
- Helps understand customer payment preference.


---

## Conclusion
This project successfully converts raw e-commerce data into meaningful insights using Python, SQL, and visualization.  
It helps understand sales performance, customer behavior, and payment trends, making it useful for business decision-making.
